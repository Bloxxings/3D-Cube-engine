# Pygame-Cube-engine

<img width="163" height="175" alt="image" src="https://github.com/user-attachments/assets/0bdc5687-9984-4938-b238-e29c9a19c554" />

This is a cube engine made with pygame and pyxel independently.
The cube can be moved and rotated.
This version is in early developpement, so be aware of bugs.

# Controls:

- Movement:  
  Z/Q/S/D : Move up, left, down, right  
  W/C : Move closer, further (Z-axis movement)

- Rotation:  
  X-axis: R/T  
  Y-axis: F/V  
  Z-axis: B/N  

- Others:  
Space: rotate cube automatically  
X: Reset the cube's position, size and direction
